Tiny Tapeout Power Gates for IHP SG13G2
=======================================

* `tt_pg_1v5_hp_1`: Single height 1.5V switch (high-performance variant)
* `tt_pg_1v5_hp_2`: Dual   height 1.5V switch (high-performance variant)
* `tt_pg_1v5_hp_4`: Quad   height 1.5V switch (high-performance variant)
* `tt_pg_1v5_ll_1`: Single height 1.5V switch (low-leakage variant)
* `tt_pg_1v5_ll_2`: Dual   height 1.5V switch (low-leakage variant)
* `tt_pg_1v5_ll_4`: Quad   height 1.5V switch (low-leakage variant)

Design and layouts were done by Sylvain Munaut.
